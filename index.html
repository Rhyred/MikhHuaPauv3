Frequently Asked Questions

Apa itu MikHuaPau?

MikHuaPau adalah alat bantu untuk bisnis RTRWNet Anda. Aplikasi ini memudahkan Anda memantau perangkat MikroTik (MikhPau) dan Huawei (HuaPau) secara real-time, serta melindungi jaringan dari serangan dengan teknologi AI.

Bagaimana cara terhubung dengan MikroTik (MikhPau)?

Untuk menghubungkan aplikasi dengan perangkat MikroTik Anda, ikuti langkah-langkah berikut:
1. Masukkan IP Public dari perangkat MikroTik Anda.
2. Gunakan port API untuk koneksi.
3. Masukkan username dan password perangkat Anda.
4. Klik 'Connect' untuk memulai pemantauan.

Apakah Huawei (HuaPau) sudah bisa terhubung dengan perangkat Huawei??

Saat ini, bagian HuaPau untuk perangkat Huawei masih dalam bentuk dummy (purwarupa) dan belum terhubung ke sistem live. Fitur ini sedang dalam pengembangan aktif dan akan segera tersedia.

Apa saja fitur utama dari deteksi DDoS berbasis AI?

Fitur deteksi DDoS kami menggunakan Machine Learning untuk memberikan perlindungan proaktif. Prosesnya mencakup pengumpulan data traffic, pelatihan model ML dengan dataset terbaru, deteksi anomali secara otomatis, dan pengiriman notifikasi push ke perangkat Anda jika terdeteksi adanya ancaman.
